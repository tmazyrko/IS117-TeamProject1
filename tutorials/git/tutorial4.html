    How to merge one branch to another

    To merge a branch into another select a branch in this case the master branch. Next click the arrow next to the branch you want to merge then simply select Merge into Current and you are done.

    Definitions

  • Master branch - The main branch of your project
  • Merge - A way to combine two different branches
