    Finding your Website

    Your AFS-hosted website(s) can be found in the public_html folder in your AFS root directory. To navigate to this folder from your root directory (for example, as soon as you log in), use the command cd public_html.

    AFS Folder Structure and Web URLs

    The public_html folder responds with the root URL for your AFS user account. For example, when we look at NJIT AFS, the /public_html folder corresponds with the public URL https://web.njit.edu/~UCID/. An index.html file placed in the /public_html folder will automatically be served when you navigate to that URL. Consequently, any subfolders within /public_html will be added on to the end of the web URL. For example, an index.html file located in /public_html/test will be served when you navigate to https://web.njit.edu/~UCID/test/.
